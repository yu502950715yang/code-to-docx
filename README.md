# 代码提取工具

本项目用于快速生成软件著作权申请所需的程序鉴别材料（源代码 Word 文档）。

## 环境准备

确保您的计算机已安装 Python 3.8+。

首先，安装项目所需的依赖库：

```bash
pip install -r requirements.txt
```

## 如何运行

### 方式一：使用 UI 界面（推荐）

这是最简单、最直观的使用方式。

1.  在项目根目录下，打开终端并运行以下命令：

    ```bash
    streamlit run app.py
    ```

2.  程序启动后，您的浏览器将自动打开一个网址（通常是 `http://localhost:8501`）。

3.  在打开的 Web 界面中，您可以：
    -   **配置项目信息**：如项目名称、版本号。
    -   **设置生成参数**：如文档页数、代码提取模式（从头尾截取或连续截取）。
    -   **管理源码路径**：添加或删除需要扫描的源代码文件夹，并指定文件类型（如 `.java`, `.vue`）。
    -   **开始生成**：点击按钮后，程序将开始工作，并实时显示进度。
    -   **下载文档**：生成成功后，页面会提供下载链接。

### 方式二：使用命令行

如果您熟悉命令行操作，也可以直接通过脚本生成文档。

-   **基本用法**：

    ```bash
    python CodeToDocx.py --pages 60 --mode start_end
    ```

-   **查看所有可用选项**：

    ```bash
    python CodeToDocx.py --help
    ```

## 项目文件结构

-   `app.py`: Streamlit UI 界面的主程序。
-   `CodeToDocx.py`: 核心的 Word 文档生成逻辑。
-   `requirements.txt`: 项目的 Python 依赖列表。
-   `README.md`: 本说明文件。
